# 灯箱白边 · FilmScan Border

把照片浮在发光的白底上 —— 模拟「底片压在灯箱(发光白台)上翻拍」的纯白透光边缘效果。

不是胶片齿孔框,而是干净的留白 + 边缘透光过曝感。上传、调节、下载,**全免费、纯前端、不上传服务器**(照片只在你浏览器本地处理)。

## 功能

- 拖拽 / 点击上传照片(JPG / PNG)
- **白边粗细**:0–25%
- **边缘透光感**:四边白光内溢,模拟灯箱过曝边缘
- **圆角 / 底色 / JPG·PNG** 微调
- 导出**原图分辨率**,不是预览缩略图

## 使用

直接在浏览器打开 `index.html` 即可,零依赖、零构建。

## 技术

单文件 HTML + Canvas,无框架、无后端。可白嫖部署到 GitHub Pages / Vercel。
