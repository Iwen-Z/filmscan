# 胶片拟真 spec — 让台面上的胶片更像真实底片

> 背景：`index.html`（单文件、原生 HTML/CSS/Canvas、零依赖）已能在观片台上自由摆放胶卷 piece、剪单张、切边框（none/film/polaroid，PR #4）。本 spec 收口「让胶片看起来更像真实底片」这一组拟真需求。
> 约束：**保持纯原生、零依赖、单文件 `index.html`**，不引入框架/构建链。复用现有 `renderPieceFilm` 的片基+齿孔+逐帧 cover+透光逻辑、`presets[]` 观片台模型、`?selftest` 自检 harness。
> 路线决定（已更新 2026-06-04）：~~只走正片~~ **改为 per-roll 卷类型**——新建卷可选 反转/黑白/负片，负片=反相+橙罩（详见 `REVAMP_SPEC.md` 阶段 D）。本文档的逐帧拟真（暗角/halation/漏光等）对三种类型**一视同仁**，都作用在画面/镜头层，不区分卷类型。
> ⚠️ 负片锁已由 REVAMP_SPEC.md 阶段D解除；负片作为 filmType=negative 的 per-roll 选项重新支持。暗角/halation/漏光等拟真效果对三种 filmType 一视同仁。

## 心智模型（物理真相，先纠一个被拧反的概念）

- 胶片**本来就是透明介质**，光恒穿透。它不是「不透光 + 一个透光旋钮」。所以片基/边框做成实心不透光是**画错了**，应一直半透明。
- 单张胶片的**透光量是固定的**，由其**密度**决定（曝光+显影出来多黑就多黑），**不可调**。
- 现实里能调的那个量，是**观片台/灯箱的背光亮度**——那是**台子的属性**，不是胶片的。当前代码把 `glow` 挂在胶片上、注释还写「透光是底片的特性」，因果拧反，需纠正。

## 阶段 0 — 透光概念纠正【已移交 → 见 REVAMP_SPEC.md 阶段 A】

> 2026-06-04 更新：阶段0 的尝试(PR #5)把背光做成「提亮胶片」是错的——胶片可摆到观片台外，背光不该提亮它。正确做法（背光只调面板亮度、胶片靠半透明片基自然透出背后台面）已并入 `REVAMP_SPEC.md` 阶段 A。**本节内容作废，以 REVAMP 阶段 A 为准。** 下方原始描述仅留档。

### （存档）原阶段 0 描述

把「透光度」从胶片属性纠正为观片台背光，片基恒定半透明。**本阶段只解决透光这一个被拧反的概念**，不碰漏光/暗角/halation。

1. **片基恒定半透明**：`renderPieceFilm` 里片基 alpha 固定为一个半透明常量（让底下背光台面透上来，边框**始终**通透），删除随 glow 变的 `baseA = 0.9 - t*0.5`。
2. **`glow` 改名 + 改归属为「背光亮度」**：
   - UI 文案「透光感」→「背光亮度」，从**胶片设置**组移到**观片台设置**组（与 unif/pixel/base 同组）。
   - 语义：背光透过胶片后的**整体提亮**强度——现每帧 `lighter` 叠加 + 边缘渐隐透光跟着它走，**并作用到片基**（背光越亮整片越亮越通透）。
   - 默认值调到肉眼明显档（解决「拖滑块没感觉」）。
3. 向后兼容（旧 piece 无字段按现状）、`?selftest` 不破、`node --check` 过。

验收：默认档下边框肉眼半透明、能看到台面透上来；拖「背光亮度」整片明显变亮/变通透。
